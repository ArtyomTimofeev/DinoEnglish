# PWA Icons

This directory should contain the following PWA icon files:

## Required Icons

- `icon-192x192.png` - 192x192px icon for Android
- `icon-512x512.png` - 512x512px icon for Android
- `apple-touch-icon.png` - 180x180px icon for iOS (optional, but recommended)
- `favicon.ico` - 32x32px favicon (optional)

## How to Generate Icons

### Option 1: PWA Asset Generator (Recommended)
Visit: https://www.pwabuilder.com/imageGenerator

1. Upload a source image (at least 512x512px, ideally square)
2. Download the generated icon pack
3. Copy the files to this directory

### Option 2: Manual Creation
Create square icons with the DinoEnglish logo or app branding:
- Use a design tool (Figma, Canva, Photoshop)
- Export as PNG files at the required sizes
- Ensure the icons have a transparent or colored background
